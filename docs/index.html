研究集会概要

本研究集会『震源インバージョンワークショップ 〜地震発生物理の包括的理解に向けた手法開発とその実践〜』では、地震時の震源過程を推定する震源インバージョン解析に携わる若手研究者が集い、震源インバージョンにまつわる諸問題および解決策について、その最新知見を共有します。 テーマを絞って密な議論を行うことで、地震発生物理の理解に資する、より良い震源インバージョン解析手法の構築可能性を模索することを目的としています。
